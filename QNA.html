Q&A

Q: How do you return to the home page from each subpage?

A: Press the back arrow on your browser or press “Computer First Aid” on top-left hand side of each page of the training manual.



Q: I want to read about Security tips for an operating system, but don’t want to scroll through the other categories. Is there a way to jump to each category?

A: Yes! At the top of each operating system page, there are links for each category for easy access.



Q: I am interested in improving/updating this training manual. Is this training manual open source?

A: Yes! All the code for this training manual is on GitHub at this link. In each HTML file, there are comments to help guide users through the code. When making changes, be sure to update the version and project history in the README file on GitHub.
